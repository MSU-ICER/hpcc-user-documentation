    BLAST+

    Basic Local Alignment Search Tool, or BLAST, is an algorithm for comparing primary biological sequence information, such as the amino-acid sequences of different proteins or the nucleotides of DNA sequences.

    https://blast.ncbi.nlm.nih.gov/

    Available modules

    The overview below shows which BLAST+ installations are available per target architecture in the HPCC module system, ordered based on software version (new to old).

    To start using BLAST+, load one of these modules using a module load command like:

    1
    module load BLAST+/2.14.1-gompi-2023a
